HTML guide #back to top

Each main file consist form sections. You can notice them by comments.

When you need to change some text in the section - open the file, find needed section and change the text that you want. You also can use option "find" in your code editor to quickly find the text.

CSS+SASS guide #back to top

All CSS files are in the folder css. Here you can find main.css file with our template styles and the vendor CSS files (bootstrap, lightbox and other).

In main.css you will find the blocks with the description. There we describe the structure of the css file.

Structure of the sass folder

In the css folder you can find the sass source for the styles. There are two folders - modules and settings.

All the sass styles are imported in the main.scss file.

JS guide #back to top

This template uses our JS file main.js and other vendor scripts such as bootstrap.min.js, jquery.min.js and other. You can find them in the js folder.

Gallery #back to top

We have Gallery section with alboms in the index.html page and the same individual page. But also we have the example of the Single Albom page (gallery-single.html). We did just one such page. But we recommend you to create it for each albom that you will have.

The images for the Gallery you can find in the img/gallery folder.
